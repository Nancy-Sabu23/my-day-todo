# my-day-todo
A modern To-Do List web application built using HTML, CSS, and JavaScript with task management, important task filtering, and local storage support.


# My Day - To-Do List App

A modern and responsive To-Do List application.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Mark tasks as important
- Filter important tasks
- Save tasks using Local Storage
- Responsive design
- Modern user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API
